Bank Queue System

Developed a Bank Queue Management System using Java to simulate real banking customer service operations. Implemented Queue data structure following the FIFO (First-In-First-Out) principle to manage customer requests efficiently. The system allows customers to join the queue, process transactions and track queue status dynamically. This project demonstrates strong understanding of Data Structures, Object-Oriented Programming and real-world system design.
